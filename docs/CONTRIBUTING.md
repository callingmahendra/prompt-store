# Contributing to Axiom Prompts

First off, thank you for considering contributing to Axiom Prompts! It's people like you that make Axiom Prompts such a great tool.

## Where do I start?

If you're looking for a place to start contributing, check out the open issues on GitHub. You can also check out the project board to see what's being worked on.

## How do I contribute?

1.  Fork the repository and clone it to your local machine.
2.  Create a new branch for your changes.
3.  Make your changes and commit them to your branch.
4.  Push your branch to your forked repository.
5.  Create a pull request to the main repository.

## Style Guide

Please follow the existing code style. We use Prettier to format our code, so please make sure you have it installed and configured in your editor.

## Questions?

If you have any questions, feel free to open an issue on GitHub.
