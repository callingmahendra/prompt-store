# Prompt Store

Prompt Store is a full-stack application designed for managing and sharing prompts. It provides a user-friendly interface to browse, create, edit, and manage prompts, with a robust backend to handle data storage and API requests.

## Getting Started

### Prerequisites

- Node.js
- npm
- Bun

### Installation and Running

**Backend:**

1.  Navigate to the `backend` directory:
    ```bash
    cd backend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

**Frontend:**

1.  Navigate to the `frontend` directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies:
    ```bash
    bun install
    ```
3.  Run the development server:
    ```bash
    bun run dev
    ```

## Technologies Used

### Backend

- **Framework:** Express.js
- **Database:** SQLite with TypeORM
- **Language:** TypeScript

### Frontend

- **Framework:** React with Vite
- **Language:** TypeScript
- **UI:** Shadcn UI, Tailwind CSS
- **Routing:** React Router
- **State Management:** TanStack Query
- **Forms:** React Hook Form
